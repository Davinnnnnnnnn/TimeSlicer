# TimeSlicer
===================================================================
                  TimeSlicer - 타임 슬라이스 이미지 생성기
===================================================================

[프로그램 소개]
TimeSlicer는 동일한 장소에서 시간의 흐름에 따라 촬영된 연속된 사진들을 
하나의 이미지로 합성해주는 도구입니다. 선형(Linear), 방사형(Radial), 
동심원(Concentric) 등 다양한 패턴으로 시간을 잘라내어 예술적인 
결과물을 만들 수 있습니다.

-------------------------------------------------------------------
1. 실행 환경 및 설치 방법
-------------------------------------------------------------------
이 프로그램은 Python 3 환경에서 동작합니다. 실행 전 아래 라이브러리가 필요합니다.

[필수 라이브러리 설치]
터미널(CMD)에서 다음 명령어를 입력하세요:
pip install opencv-python numpy PyQt5

[실행 방법]
python time_slicer.py

-------------------------------------------------------------------
2. 기본 사용 순서 (Quick Start)
-------------------------------------------------------------------
1) [Open Folder] 버튼을 눌러 사진들이 들어있는 폴더를 선택합니다.
   (지원 포맷: jpg, jpeg, tif, png)
2) 왼쪽 목록에서 합성할 사진들을 선택합니다.
   (기본적으로 모두 선택되지만, 필요한 구간만 체크할 수 있습니다.)
3) 오른쪽 패널에서 [Pattern]을 선택하고 모양을 조절합니다.
   - 화면에 보이는 가이드 라인(선)이 사진이 잘리는 경계면입니다.
4) [Show Output Preview] 버튼을 눌러 결과물을 미리 확인합니다.
5) 설정이 만족스럽다면 [Generate High-Res TIFF]를 눌러 고화질 파일로 저장합니다.

-------------------------------------------------------------------
3. 주요 기능 및 옵션 설명
-------------------------------------------------------------------

[1] 패턴 (Pattern) 종류
   - Linear (선형): 한쪽 끝에서 반대쪽으로 시간이 흐릅니다. (일출/일몰에 적합)
   - Radial (방사형): 시계 바늘처럼 빙글빙글 돌아가는 형태입니다.
   - Concentric (동심원): 중심에서 바깥쪽으로 퍼져나가는 형태입니다.

[2] 중심점 설정 (Center Position)
   - Radial, Concentric 모드에서 중심점이 중요합니다.
   - 우측의 X, Y 슬라이더를 움직이거나, **이미지 화면을 직접 클릭**하면
     그 위치가 중심점으로 설정됩니다.

[3] 블렌딩 (Blending) - 경계선 부드럽게 하기
   - Blend Amount: 사진 조각 간의 경계선을 흐릿하게 뭉갭니다.
     수치가 높을수록 경계가 사라지고 부드러워집니다.
   - Blend Linearity: 블렌딩의 강도 곡선을 조절합니다.

[4] 파일 선택 도구 (Auto Selection)
   - 사진이 너무 많을 경우(예: 타임랩스 1000장), 'Auto...' 버튼을 누르세요.
   - "Every N-th File": 5장마다 1장씩 선택 등 간격을 둡니다.
   - "Total N Files": 전체 중 딱 50장만 균일하게 뽑아냅니다.

-------------------------------------------------------------------
4. 화면 가이드라인 보는 법
-------------------------------------------------------------------
- 화면에 표시되는 선들은 사진이 바뀌는 '시간의 경계선'을 의미합니다.
- **굵은 색(또는 다른 색) 선**: 첫 번째 사진(시작점)이 배치되는 위치입니다.
- 가이드라인이 너무 많아 어지럽다면 'View Less Lines'를 체크하세요.
- 가이드라인 색상이 사진과 비슷해 안 보이면 [Guide Color] 버튼으로 변경하세요.

-------------------------------------------------------------------
5. 주의사항 (필독)
-------------------------------------------------------------------
1) 삼각대 필수: 
   카메라가 고정된 상태에서 찍은 사진이어야 결과물이 자연스럽습니다. 
   배경이 흔들렸다면 결과물이 어긋나 보일 수 있습니다.

2) 사진 순서: 
   파일 이름 순서대로 시간이 흐른다고 가정합니다. 
   파일명이 뒤죽박죽이면 결과물도 뒤섞입니다.

3) 미리보기 vs 실제 저장:
   프로그램 내 'Preview'는 속도를 위해 화질을 낮춰서 보여줍니다.
   [Generate] 버튼으로 저장하는 최종 결과물은 원본 화질을 유지하므로,
   생성 시간이 다소 걸릴 수 있습니다 (특히 고화질 TIFF 저장 시).

4) 메모리 사용:
   수천 장의 고화질 이미지를 한 번에 처리할 때는 PC의 메모리(RAM)를 
   많이 사용할 수 있습니다.
